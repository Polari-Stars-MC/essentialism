# 本质主义 Essentialism

一个以“九大本质”驱动世界认知、采集、精炼与重构的 Minecraft Mod 设计案。

## 项目概览

`本质主义（Essentialism）` 试图回答一个问题：

> 如果 Minecraft 世界里的每一种方块、物品、生物与现象，都可以被拆解为更底层的“本质”，那么玩家能否像研究自然法则一样，理解这个世界、操纵这个世界，甚至重新书写这个世界？

这个项目不是简单增加一批新矿物或设备，而是围绕“解析 -> 提取 -> 储存 -> 重构 -> 共振”建立完整玩法循环。

它的目标是同时提供：

- 清晰的资源成长线
- 可推演的底层规则
- 具有辨识度的装置系统
- 带有哲学气质的文案与氛围

## 设计目标

### 1. 让“世界认知”成为玩法的一部分

玩家不是立刻获得答案，而是通过扫描、比对、实验逐渐理解九种本质的分布规律和相互关系。

### 2. 让资源系统有“化学感”

不同方块不是掉落固定材料，而是携带可被解析、分离、浓缩、重组的本质配比。玩家面对的不只是配方，而是一套可组合的底层语言。

### 3. 让后期玩法摆脱单纯数值堆叠

后期成长不只体现在更强装备，而体现在：

- 对区域本质的调控
- 对物品属性的重构
- 对环境规则的短暂改写
- 对悖论物品与隐藏机制的探索

### 4. 保持 Minecraft 原生语义

九大本质的来源与行为应尽量从原版经验中“自然生长”出来，而不是完全脱离 Minecraft 语境另起一套世界观。

## 核心玩法循环

完整循环建议分为五步：

1. **解析**：用透镜或设备读取方块/物品的本质构成。
2. **提取**：从目标对象中提炼出高浓度本质。
3. **储存**：将本质转化为可运输、可堆叠、可计算的介质。
4. **重构**：利用本质配方制造新物品，或修改已有物品属性。
5. **共振**：在区域尺度操控本质总量，触发环境级效果。

这个循环对应玩家的认知成长：

- 前期学会“看见”
- 中期学会“分离”
- 后期学会“改写”
- 终局学会“支配规则”

## 九大本质详解

### 坚固 - 存在的锚点

*“我是基岩的叹息，是钻石的傲慢。”*

- **象征**：硬度、耐久、抗性、不变
- **来源**：石头、矿物、金属、盔甲、高硬度方块
- **高阶应用**：锻造不毁之器、加固建筑、抵御爆炸
- **设计价值**：为装备改造、防御工事、环境抗性提供支撑
- **哲学注脚**：坚固者的悲剧在于，世界塌陷时，他们总是最后才知道。

### 生命 - 生长的悖论

*“你的每一次盛开，都只是发给我的邀请函。”*

- **象征**：生长、治愈、繁殖、循环
- **来源**：草、原木、作物、动物、药水效果
- **高阶应用**：催熟作物、治愈不死生物、创造生命体
- **设计价值**：服务农业、回复、生态互动与生长型自动化
- **哲学注脚**：生命是一场与熵增的共谋。活着，就是在缓慢地死去。

### 熵增 - 衰败的必然

*“你们的狂欢，终归是我的遗产。”*

- **象征**：腐朽、混乱、损耗、终结
- **来源**：沙子、苔石、腐肉、亡灵生物、爆炸
- **高阶应用**：加速老化、溶解物品、引发局部终结
- **设计价值**：作为高风险高收益本质，驱动分解、腐蚀、代价机制
- **哲学注脚**：熵增从不催促。它只是等待，因为时间站在它那边。

### 光明 - 揭示的暴力

*“你的黑暗不过是我的幕布。”*

- **象征**：辐射、热量、信息、显现
- **来源**：火把、岩浆、荧石、附魔光效、发光方块
- **高阶应用**：透视墙体、灼烧暗影生物、制造真相之域
- **设计价值**：强化侦测、净化、区域压制与视觉反馈
- **哲学注脚**：光明从不照亮黑暗，它只是收回了自己的影。

### 暗影 - 隐匿的仁慈

*“没有你，我只是一片平等的虚无。”*

- **象征**：隐藏、吸纳、空洞、湮没
- **来源**：虚空、黑曜石、末影珍珠、低亮度区域
- **高阶应用**：隐身、存储虚空、吞没光线
- **设计价值**：适合做潜行、规避、删除式破坏与特殊存储
- **哲学注脚**：暗影不是光的反面。光是暗影的伤口。

### 运动 - 变化的冲动

*“停下？停下就是死亡！”*

- **象征**：位移、动能、振动、速度
- **来源**：矿车、活塞、水流、箭矢、玩家移动
- **高阶应用**：瞬间移动、动能武器、动力结构
- **设计价值**：连接红石机械、移动能力与战斗位移
- **哲学注脚**：静止是运动的特例，正如死亡是生命的特例。

### 心灵 - 意识的牢笼

*“没有一双眼睛，光明算什么光明？”*

- **象征**：意志、记忆、经验、感知
- **来源**：村民交易、经验球、附魔书、头颅、书架
- **高阶应用**：读取方块记忆、植入思想、制造虚假经验
- **设计价值**：适合附魔、经验系统、智能装置与叙事事件
- **哲学注脚**：心灵是一座监狱，囚禁者既是狱卒，也是囚徒。

### 时空 - 折叠的沉默

*“你们争论的都是‘一物’，而我管理的是‘万物之间’。”*

- **象征**：距离、维度、瞬移、延迟
- **来源**：下界传送门、紫颂果、潜影贝、高稀有度方块
- **高阶应用**：扭曲空间、暂停时间、跨维度传送
- **设计价值**：承担后期机动、区域控制、维度交互和高级配方
- **哲学注脚**：时空从不发言。它只是悄悄地把“这里”变成“那里”。

### 共鸣 - 连接的秘密

*“没有我，你们只是独奏。”*

- **象征**：同步、传递、共振、反应
- **来源**：音符盒、红石信号、繁殖行为、成对物体
- **高阶应用**：范围叠加效应、信号无线传输、群体意志
- **设计价值**：是自动化、网络化装置、区域联动的底层纽带
- **哲学注脚**：共鸣是最安静的暴力，它让你以为“在一起”是你自己的决定。

## 本质之间的相互作用

| 本质A | 本质B | 互作类型 | 结果示例                 |
|-----|-----|------|----------------------|
| 坚固  | 熵增  | 相克   | 加固物品会减缓老化；老化会削弱坚固    |
| 生命  | 熵增  | 相生相克 | 生命的存在以熵增为代价；死亡为新生让路  |
| 光明  | 暗影  | 互相定义 | 一方越强，另一方的边界越清晰       |
| 运动  | 时空  | 竞争   | 极致运动会撕裂时空；折叠时空会取消运动  |
| 心灵  | 共鸣  | 放大   | 心灵通过共鸣扩散；共鸣需要心灵锚定    |
| 坚固  | 时空  | 对抗   | 基岩的“无限坚固”来自它对时空的拒绝   |
| 生命  | 心灵  | 不可通约 | 草有生命而无心灵；末影人有心灵而难言生命 |
| 熵增  | 暗影  | 同谋   | 衰败之物趋向隐匿；隐匿之物加速衰败    |
| 共鸣  | 运动  | 协作   | 红石信号驱动活塞；声音触发机关      |

### 设计建议

为了让系统更可玩，建议后续实现时遵循三条原则：

- **互作关系要少而硬**：优先保证少数关系稳定可推演，而不是所有本质都两两复杂反应。
- **互作要能被玩家观察到**：粒子、音效、方块状态、UI 数值都应明确反馈。
- **互作必须服务玩法决策**：不是只写在设定里，而是能影响采集、战斗、建造或自动化。

### 数值实现建议

后续如果进入开发，推荐把本质数值控制在两个层级：

- **设计层**：对外展示使用 `无 / 微 / 低 / 中 / 高 / 极高 / 无限`

这样能同时兼顾：

- 玩家认知负担不过高
- 系统内部可精确计算
- UI 表达可做分层显示

## 特殊机制

### 本质共振（区域效应）

当同一区块内某种本质的总和超过阈值时，会触发区域变异：

| 本质 |   阈值 | 区块变异效果                 |
|----|-----:|------------------------|
| 坚固 | 5000 | 所有方块硬度 +1，挖掘时间 +20%    |
| 生命 | 4000 | 植物生长速度 x2，动物繁殖冷却减半     |
| 熵增 | 3000 | 物品耐久损耗 +50%，食物腐烂加速     |
| 光明 | 3500 | 禁止自然怪物生成，玩家获得夜视        |
| 暗影 | 3500 | 光照等级 -5，幻翼生成概率提高       |
| 运动 | 4000 | 玩家移速 +10%，水流速度 x2      |
| 心灵 | 2500 | 经验球掉落 +50%，村民交易折扣 +10% |
| 时空 | 2000 | 随机生成紫颂粒子，偶尔触发短暂传送异响    |
| 共鸣 | 3000 | 红石信号传输无衰减，音符盒音域 +1     |

### 本质枯竭（过度采集惩罚）

同一方块被反复提取本质超过 5 次后，会进入 **枯竭状态**：

- 外观出现裂纹粒子
- 本质产出降至 10%
- 约 3 个 Minecraft 日后自然恢复
- 枯竭期间若继续提取，有概率永久消失

这个机制的意义在于：

- 抑制单点无限刷取
- 鼓励玩家迁移与探索
- 为区域管理与恢复玩法留空间

### 本质异变（稀有变异体）

挖掘任何方块时，有 **0.5%** 的概率掉落该方块的 **本质异变体**：

| 异变体类型 | 效果              | 示例                  |
|-------|-----------------|---------------------|
| 纯净体   | 只包含单一本质，浓度 +50% | 纯净坚固水晶（坚固 140）      |
| 共鸣体   | 共鸣翻倍，其他减半       | 共鸣草方块（共鸣 80）        |
| 虚空体   | 大量暗影 + 时空，其他归零  | 虚空钻石（暗影 60 + 时空 60） |
| 生命悖论体 | 生命与熵增同时极高       | 悖论苹果（生命 70 + 熵增 70） |
| 时间琥珀  | 冻结方块被破坏时的状态     | 时间琥珀草（放置后恢复原貌）      |

## 核心装置

### 解析者透镜（手持）

- **合成**：紫水晶碎片 + 铜锭 + 玻璃板
- **功能**：
  - 右键方块，聊天栏显示本质摘要
  - 潜行 + 右键，显示更详细的光谱分布
  - 可附魔：`精准`、`范围`
- **能量**：无需能量，无限使用
- **定位**：前期认知工具，是整个系统的第一把钥匙

### 简易解析器（方块）

- **合成**：透镜 + 铁锭 + 红石比较器
- **界面**：
  - 单格输入槽
  - 九条彩色进度条
  - 百分比数值显示
- **处理时间**：2 秒，可受红石加速
- **能量**：消耗红石信号，强度影响速度
- **定位**：将“手持观察”升级为“稳定分析”

### 进阶光谱仪（方块）

- **合成**：简易解析器 + 末影珍珠 + 下界石英块
- **界面**：
  - 3x3 批量扫描槽
  - 九种本质折线图
  - 可导出数据到本质储存单元
- **处理时间**：1 秒/格，并行处理
- **特殊功能**：
  - 识别稀有变异体
  - 预测本质共振阈值
  - 显示区块热力图
- **定位**：中后期信息设备

### 灵魂离心机（方块）

- **合成**：下界合金锭 + 幽匿催发体 + 灵魂沙
- **功能**：
  - 将混合本质物品分离为纯净本质溶液
  - 输入为带有多种本质的物品
  - 输出 1-3 种溶液瓶
- **界面**：
  - 1 格输入槽
  - 3 格输出槽
  - 九种本质分离进度条
- **处理时间**：10 秒，受复杂度影响
- **能量**：消耗经验等级（建议 5 级/次）
- **定位**：把“看见”变成“可利用”

### 符文矩阵编纂桌（方块）

- **合成**：下界合金块 + 附魔台 + 紫水晶块
- **功能**：
  - 将本质溶液重构成新物品
  - 支持数据包扩展配方
  - 可修改已有物品的本质属性
- **界面**：
  - 3x3 本质网格
  - 1 格催化剂槽
  - 1 格输出槽
  - 最近 10 条配方记忆
- **处理时间**：瞬时结算，确认后消耗
- **能量**：消耗经验值，建议与配方复杂度挂钩
- **定位**：中后期核心生产台

### 本质共鸣矩阵（多方块结构）

- **构建**：`3x3x3` 紫水晶块框架 + 中央幽匿催发体
- **功能**：
  - 实时监测 `16x16x16` 区域本质波动
  - 达到阈值时自动触发事件
  - 可连接红石输出
- **设置项**：
  - 监测本质类型
  - 触发阈值 `0-10000`
  - 触发动作（红石 / 命令 / 粒子）
- **范围**：以矩阵为中心半径 32 格
- **能量**：持续消耗共鸣本质
- **定位**：终局阶段的区域控制装置

### 配方系统建议

为了后续可扩展，推荐把配方拆成三类：

- **分离配方**：物品 -> 本质溶液
- **转化配方**：物品 + 本质 -> 变体物品
- **重构配方**：多本质 + 催化剂 -> 新物品

如果未来走数据驱动，建议配方定义最少包含：

- 输入对象
- 本质需求
- 最低浓度
- 催化剂
- 产物
- 失败条件
- 副作用

## 游戏阶段与目标

### 初始阶段 - 发现本质

- 合成 **解析者透镜**
- 扫描常见方块，建立认知
- 目标：认识九种本质的基础分布规律
- 里程碑：扫描 50 种不同方块

### 进阶阶段 - 掌握精炼

- 合成 **简易解析器** 与 **灵魂离心机**
- 开始提取和储存本质
- 目标：获得第一批高纯度本质溶液
- 里程碑：成功分离任意一种 `60+` 浓度本质

### 深化阶段 - 本质重组

- 合成 **进阶光谱仪** 与 **符文矩阵编纂桌**
- 尝试特殊工具和环境型道具
- 目标：做出第一件本质重构物品
- 里程碑：制作“虚空之刃”或“时间之桶”

### 巅峰阶段 - 本质大师

- 构建 **本质共鸣矩阵**
- 操控区域级共振
- 探索异变体与隐藏配方
- 目标：达成九种本质的“无限”浓度各一次
- 里程碑：触发全部 9 种区块共振效果

### 终局挑战 - 悖论合成

隐藏成就或挑战可以围绕以下三种悖论物品展开：

1. **永恒之烬**：生命 100 + 熵增 100。放置后持续燃烧，但火焰不会熄灭也不会蔓延。
2. **光明斗篷**：光明 100 + 暗影 100。穿戴后隐身，但脚下会留下发光足迹。
3. **静止引擎**：运动 100 + 时空 100。范围内实体无法移动，但仍可旋转和攻击。

### 文案风格示例

**解析者透镜**

> *“紫水晶凝视世界的裂缝，铜接纳流动的真理，玻璃承诺不被扭曲。当你透过它看见草方块内部的绿光时，你会明白 世界从未向你隐藏，你只是从未学会观看。”*

**灵魂离心机**

> *“幽匿催发体低语着分离的仪式。下界合金的骨架囚禁着暴烈的能量。灵魂沙叹息着释放被混合的自我。在这里，草与石被剥去伪装，露出它们真正的颜色。”*

**时间之桶**

> *“水是流动的遗忘。时空是折叠的记忆。当它们相遇，时间学会了犹豫。倒出的水会犹豫要不要向前，因为它们记得自己曾经来过这里。”*

**虚空之刃**

> *“剑不再是切割，而是删除。被它触碰的不是伤口，是坐标的空白。被它击中的敌人不会死去，他们只是 不再属于这里。”*

### 风格约束建议

为了避免文案失控，后续写作可统一遵循：

- 句子尽量短，意象明确
- 哲学感服务氛围，不替代规则说明
- 装置说明分为“功能描述”和“风格描述”两层
- UI 文案偏清晰，物品描述偏诗性
